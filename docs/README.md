## 1 关于本站

这是一个部署在本地电脑上的网站，只有处于公司内网时可以访问。主要用途包括：
- 收集和分享有用的互联网文章、软件资源；
- 发布产品工作文档；
- 记录一些需要频繁使用的站点、工具等资料。

## 2 网站维护
本网站所有文章均使用markdown语法编写而成。

- [下载简单易用的markdown文档编辑器：typora](https://www.typora.io/)
- [markdown语法的简易教程](https://www.runoob.com/markdown/md-tutorial.html)
